Blog Tech Stack

As I’m looking to start up a mailing service I thought it be interesting to talk about the services I’m using for hosting this blog. I’ll do it in chronological order. Kind of.

Service Type Software/Provider
Static Site Generator Pelcian
Static Site Host Github Pages
Comments/Forum Discourse
Dynamic Content Host [1] Digital Ocean
Domain Registration/DNS Google Domains
Transactional Email [2] Mailgun
Mailing List Mailchimp
[1]Used for hosting the discourse instance
[2]Transactional email is, according to Mailchimp’s Docs, an email sent to an individual based on some action. Discourse uses it for signups, which is why I needed it.
